# 부동산 특약 도우미 개인정보 처리방침

**부동산 특약 도우미**(이하 '본 서비스')는 정보주체의 자유와 권리 보호를 위해 「개인정보 보호법」 및 관계 법령이 정한 바를 준수하여, 적법하게 개인정보를 처리하고 안전하게 관리하고 있습니다. 이에 「개인정보 보호법」 제30조에 따라 정보주체에게 개인정보의 처리와 보호에 관한 절차 및 기준을 안내하고, 이와 관련한 고충을 신속하고 원활하게 처리할 수 있도록 하기 위하여 다음과 같이 개인정보 처리방침을 수립·공개합니다.

---

## 1. 개인정보의 처리 목적

본 서비스는 다음의 목적을 위하여 개인정보를 처리합니다. 처리하고 있는 개인정보는 다음의 목적 이외의 용도로는 이용되지 않으며, 이용 목적이 변경되는 경우에는 「개인정보 보호법」 제18조에 따라 별도의 동의를 받는 등 필요한 조치를 이행할 예정입니다.

1. **부동산 특약 추천 서비스 제공**
   - 사용자가 업로드한 임대차계약서(PDF/이미지) 분석
   - 사용자가 입력한 질문·특약 텍스트·논쟁 상황에 대한 AI 기반 답변 생성
   - 관련 법령 근거 및 판례 정보 제공
   - 특약 문구 제안

> 본 서비스는 회원 가입 기능이 없으며, 별도의 계정 정보를 수집하지 않습니다.

---

## 2. 처리하는 개인정보의 항목

본 서비스는 서비스 제공을 위해 필요한 최소한의 개인정보만을 처리하며, 처리하는 개인정보 항목은 다음과 같습니다.

| 구분                            | 수집 항목                                                                                   | 수집 방법               |
| ------------------------------- | ------------------------------------------------------------------------------------------- | ----------------------- |
| 서비스 이용 시 사용자 입력 정보 | 업로드된 임대차계약서 텍스트 (PDF/이미지에서 추출), 채팅 질문 내용, 직접 작성한 특약 텍스트 | 사용자 직접 입력·업로드 |

> **주의**: 임대차계약서에는 당사자 성명, 주소, 주민등록번호 등 개인정보가 포함될 수 있습니다. 본 서비스는 계약서 내용 중 특약 분석에 필요한 텍스트만을 처리하며, 계약서에 포함된 개인식별정보를 별도로 추출·저장하지 않습니다.

---

## 3. 개인정보의 처리 및 보유 기간

본 서비스는 개인정보를 별도로 저장하지 않으며, 다음과 같이 처리합니다.

| 처리 항목              | 보유 기간              | 비고                          |
| ---------------------- | ---------------------- | ----------------------------- |
| 업로드된 계약서 텍스트 | 세션 종료 시 즉시 삭제 | 서버에 영구 저장하지 않음     |
| 채팅 질문 및 응답 내용 | 세션 종료 시 즉시 삭제 | 브라우저 세션 메모리에만 존재 |

- 본 서비스는 데이터베이스에 사용자 입력 정보를 저장하지 않습니다.
- 브라우저 창을 닫거나 세션이 종료되면 모든 입력 데이터는 즉시 삭제됩니다.

---

## 4. 개인정보의 파기 절차 및 방법

본 서비스는 개인정보 보유 기간의 경과, 처리 목적 달성 등 개인정보가 불필요하게 되었을 때에는 지체 없이 해당 개인정보를 파기합니다.

1. **파기 절차**: 세션 종료 시 서버 메모리에서 자동으로 삭제됩니다. 별도의 데이터베이스에 이전·보관하지 않습니다.
2. **파기 방법**: 전자적 파일 형태로 처리된 정보는 메모리에서 즉시 해제되어 재생할 수 없도록 처리합니다.

---

## 5. 개인정보의 제3자 제공에 관한 사항

본 서비스는 원칙적으로 정보주체의 개인정보를 제3자에게 제공하지 않습니다.

다만, AI 응답 생성을 위해 다음과 같이 외부 API를 활용할 수 있으며, 이 경우 사용자의 질문 내용이 해당 API 서비스로 전송됩니다.

| 제공받는 자                | 제공 목적                       | 제공 항목                              | 보유 및 이용 기간                                               |
| -------------------------- | ------------------------------- | -------------------------------------- | --------------------------------------------------------------- |
| LLM API 제공사 (추후 확정) | 사용자 질문에 대한 AI 응답 생성 | 사용자 질문 텍스트, 계약서 추출 텍스트 | API 처리 완료 즉시 삭제 (API 제공사의 개인정보 처리방침에 따름) |

> 해당 LLM API 제공사가 확정되면 구체적인 서비스명과 해당 제공사의 개인정보 처리방침 링크를 본 항목에 업데이트합니다.

## 6. 개인정보 처리업무의 위탁에 관한 사항

본 서비스는 현재 개인정보 처리업무를 외부에 위탁하고 있지 않습니다.

향후 위탁이 발생하는 경우 위탁받는 자와 위탁 업무의 내용을 본 개인정보 처리방침에 공개하겠습니다.

## 7. 개인정보의 안전성 확보조치에 관한 사항

본 서비스는 개인정보의 안전성 확보를 위해 다음과 같은 조치를 취하고 있습니다.

1. **기술적 조치**
   - 사용자 입력 데이터는 서버 메모리에서만 처리되며, 별도의 데이터베이스에 영구 저장하지 않습니다.
   - HTTPS를 통한 데이터 전송 암호화를 적용합니다 (배포 시).
   - 세션 종료 시 모든 사용자 데이터를 자동 삭제합니다.

2. **관리적 조치**
   - 개인정보에 접근할 수 있는 담당자를 최소화하고 있습니다.

1. 본 서비스는 별도의 회원 가입 없이 이용 가능하며, 사용자 데이터를 영구 저장하지 않으므로 세션 종료 시 모든 데이터가 자동 삭제됩니다.
